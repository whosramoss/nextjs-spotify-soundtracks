## How to install 

```bash
  # Clone the project
  git clone https://github.com/whosramoss/nextjs-spotify-soundtracks/

  # Go to the project directory
  cd nextjs-spotify-soundtracks

  # Install dependencies
  npm install

  # Start the server 
  npm run dev
```
